#Environment setting
1. Python that we are using is 2.7
2. pip install django, tweepy, scikit-learn

#How to Run the code
1. Download or clone to local
2. Go the directory
3. In cmd or Terminal, run 'python manage.py migrate' first, then 'python manage.py runserver'
